Welcome

Üdvözlöm

Backend fókuszú szoftverfejlesztő és technikai vezető vagyok, erős C# és .NET háttérrel. Komplex rendszerek tervezésével és továbbfejlesztésével, legacy kódbázisok modernizálásával és a valós működési korlátokat figyelembe vevő architekturális döntésekkel foglalkozom.

A fejlesztés mellett csapatot vezetek koordináció, mentorálás és code review révén, valamint Scrum Masterként támogatom a hatékony és fenntartható munkavégzést.

Ez az oldal gyakorlati mérnöki döntéseket, backend mintákat és éles rendszerekből származó tapasztalatokat mutat be, természetesen maximális diszkrécióval.
